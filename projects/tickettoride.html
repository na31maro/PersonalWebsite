Ticket to Ride CSIS-225 Object Oriented Programming Final

Note: It was recommended that this project be created using a Java Applet. Java Web Programs were deemed insecure and abandoned in HTML5, and also abondened in Blue J version 4.0. In order to showcase this project, the optimal way to do so was using a video.

Project Description

This assignment is the "make it or break it" project for Computer Science students at Siena College. Every programming student has several stories to tell about their time coding Ticket to Ride.
Students are thrown into groups of 5 or 6 with extremely minimal knowledge on graphical programming, version control and coding in groups. These are all essential skills to the project and you need to develop them fast in order to succeed.

Project Details

  • Create a Java program that simulates the popular board game Ticket to Ride
  • Work in a group of 5 or 6
  • Learn how to effectively version control
  • Develop graphical programming skills
