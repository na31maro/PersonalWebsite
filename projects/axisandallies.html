Project Description

Your task is to create a simulation, for statistical gathering purposes of general combat for the board game Axis & Allies 1942, Second Edition. This simulation will answer the following question: what is the probabilty that my forces can defeat and take over a territory defended by a certain amount of defending forces.

Project Details

  • Official Axis and Allies Rules
  • Create a java program that will simulate Axis and Allies combat
  • Design and run simulation test cases to answer reflection questions exploring the best combat tactics
